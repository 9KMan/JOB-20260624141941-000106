# Out of Scope

The following items are explicitly NOT included in this build:

## Out of Scope
- Mobile/tablet versions (desktop only)
- Web deployment (PySide6 desktop only)
- Multi-user authentication (single-user app)
- Cloud LLM providers other than AWS Bedrock
- OCR for scanned PDFs (relies on text-layer PDFs)
- Multi-language UI (English only)
- Real-time collaboration features

## Future Phases
- Distribution via PyInstaller/briefcase
- Auto-update mechanism
- Plugin system for custom prompts
- Settings sync across machines
- Versioning of generated reports
- Template marketplace for prompts
